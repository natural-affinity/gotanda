# Gotanda (五反田)

Test helpers for golang projects.

## Dependencies
* [makefiles](https://github.com/natural-affinity/makefiles)
* `make watch`: [kouhai](https://github.com/natural-affinity/kouhai)

## Usage
```bash
go doc gotanda <func|type>
```
